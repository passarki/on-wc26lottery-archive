# on-wc26lottery-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，可直接通过浏览器访问。

## 项目简介

`on-wc26lottery-archive` 是一个以归档为目的的仓库，主要存放与特定活动或事件相关的 HTML 页面。所有页面均为独立文件，不依赖后端服务或外部数据库。

## 目录说明

```
on-wc26lottery-archive/
├── index.html          # 仓库入口页面
├── pages/              # 存放归档页面的文件夹
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 静态资源文件（可选）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的默认入口页面，通常包含导航或说明。
- `pages/`：存放所有归档的 HTML 页面文件，每个文件对应一个独立的归档内容。
- `assets/`：存放页面引用的样式、脚本或图片等资源（如存在）。

## 页面归档说明

本仓库中的 HTML 页面均为历史存档，保留原有结构和内容。页面可能包含简化的交互或展示功能，但不会主动收集用户信息或进行网络请求。

归档页面采用静态 HTML + CSS + JavaScript 实现，可在现代浏览器中直接运行。部分页面可能需要启用 JavaScript 才能正常显示。
